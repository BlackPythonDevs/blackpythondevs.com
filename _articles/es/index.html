¿Qué es Black Python Devs?

Black Python Devs es una comunidad online-first para desarrolladores de Python de todos los niveles de habilidad que se identifican como negros.

Nuestro objetivo es:

  1. Crea un sentido de comunidad entre los Desarrolladores Negros de todo el mundo.
  2. Apoye a los desarrolladores de Black Python con recursos de desarrollo profesional.
  3. Compartir las experiencias de otros grupos con comunidades de Desarrolladores Negros.
